# Projeto-01m
Criação de um caderno temático usando a ferramenta NotebookLM

## Objetivo do Projeto
O objetivo deste projeto foi analisar e sintetizar informações a partir de diferentes documentos, utilizando o NotebookLM como ferramenta de apoio ao aprendizado.

## Utilização do NotebookLM
O NotebookLM foi utilizado como ferramenta central para:
- Organizar os documentos de estudo em um único ambiente;
- Fazer perguntas direcionadas sobre os conteúdos;
- Gerar resumos automáticos;

## Fontes e Documentos Utilizados
Os seguintes materiais foram utilizados como base para as consultas:
- Artigos acadêmicos em PDF;
- Sites com tutoriais sobre o tema;
- Videos explicativos do Youtube;

## Processo de Análise e Aprendizado
O processo de análise ocorreu em etapas:
1. Leitura inicial dos documentos para entendimento geral;
2. Inserção dos materiais no NotebookLM;
3. Formulação de perguntas específicas sobre os temas principais;
4. Análise das respostas e resumos gerados pela ferramenta;

## Principais Resultados e Insights
Como principais resultados, foi possível:
- Obter uma visão mais clara e organizada do conteúdo estudado;
- Identificar conceitos centrais e relações entre diferentes fontes;
- Economizar tempo na leitura e revisão dos materiais;
- Desenvolver uma compreensão mais crítica e aprofundada do tema.
